## Design Styles

### 1. Clean Card-Based Dashboard (Recommended)

**Characteristics:**
- Generous white space
- Rounded corners (12-16px radius)
- Subtle shadows (0 2px 8px rgba(0,0,0,0.08))
- Clear visual hierarchy with card groupings
- Progress bars and circular progress indicators
- Minimal iconography (outline style, 24px)

**Typography:**
- Headers: Inter or Plus Jakarta Sans (600-700 weight)
- Body: Inter (400-500 weight)
- Scale: 14px base, 1.5 line height

**Best for:** Data-heavy dashboards, assessment results, learning paths  
**Reference feel:** Notion, Linear, Headspace dashboard

### 2. Soft Neumorphic

**Characteristics:**
- Soft inset/outset shadows creating depth
- Monochromatic backgrounds with subtle gradients
- Rounded, pillow-like buttons and inputs
- Floating card effect
- Progress rings with gradient strokes

**Typography:**
- Headers: Outfit or Manrope (500-700 weight)
- Body: Outfit (400 weight)
- Slightly larger touch targets for mobile

**Best for:** Onboarding flows, goal setting, reflection journals  
**Caution:** Requires careful contrast management for accessibility

### 3. Minimalist Clinical

**Characteristics:**
- Sharp, clean lines
- Minimal decoration—content-first
- Strong typography hierarchy
- Data visualization focus (radar charts, line graphs)
- Accent color used sparingly for actions only
- High contrast, WCAG AAA compliant

**Typography:**
- Headers: DM Sans or IBM Plex Sans (600 weight)
- Body: DM Sans (400 weight)
- Compact scale: 13px base

**Best for:** Assessment interfaces, analytics, professional credentialing  
**Reference feel:** Medical dashboards, Stripe dashboard
